# Minta Maaf ke Kamu 🥺💖

Sebuah web interaktif yang manis untuk minta maaf ke orang terkasih dengan sentuhan personal!

## Fitur ✨

- 💕 **Input Tanggal Lahir** - Masukkan tanggal lahir untuk personalisasi pesan
- 🥺 **Pesan Maaf yang Touching** - Pesan apology yang dipersonalisasi
- 💖 **Animasi Hati** - Hati-hati cantik yang mengapung
- 📱 **Responsive Design** - Berjalan sempurna di mobile dan desktop
- 🎨 **UI/UX Cantik** - Desain yang sweet dan romantic

## Cara Menggunakan 📖

1. Buka `index.html` di browser Anda
2. Masukkan tanggal lahir sayang
3. Klik tombol "Lanjut 💌"
4. Pilih "Maafin 💖" atau "Enggak 😠"
5. Nikmati animasi dan pesan maaf yang special!

## Struktur File 📁

```
di-baca-ya-sayang/
├── index.html      # File HTML utama
├── style.css       # Styling dan animasi
├── script.js       # Logika interaktif
└── README.md       # File dokumentasi ini
```

## Customization 🎨

Anda bisa customize pesan maaf dengan mengubah text di dalam `script.js`:

```javascript
apologyText.textContent = `Maaf ya sayangg, pesan kustom Anda di sini!`;
```
